# Momentum
Приложение, подобное оригинальному из магазина хрома.
Функционал: 
- виджет погоды 
- музыкальный плеер
- слайдер изображений (при клике случайным образом выбирается 1 из 20 изображений в зависимости от времени суток. То есть для каждого периода суток (день, утро и т.д.) 20 изображений)
- виджет цитата дня. В зависимости от даты показывается цитата о значимом событии в истории в этот день. 

Приложение на чистом JS без использования сторонних библиотек.
